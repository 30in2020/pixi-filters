# EmbossFilter

PixiJS v4 filter to render DisplayObject as ASCII text.

## Installation

```bash
npm install @pixi/filter-emboss
```

## Usage

```js
import {EmbossFilter} from '@pixi/filter-emboss';

const container = new PIXI.Container();
container.filters = [new EmbossFilter()];
```

# ShockwaveFilter

PixiJS v4 filter to render DisplayObject as ASCII text.

## Installation

```bash
npm install @pixi/filter-ascii
```

## Usage

```js
import {ShockwaveFilter} from '@pixi/filter-ascii';

const container = new PIXI.Container();
container.filters = [new ShockwaveFilter()];
```
